⚙️ 管理員功能詳細操作說明

快速上手指南 > 管理員功能
1️⃣ 進入管理員專區
首頁 → 管理員專區

🔐 權限確認

  • 只有管理員角色才能進入管理員專區
  • 登入後系統會顯示您的角色權限
  • 一般會員無法看到或點擊管理員專區按鈕
  • 確認您的角色為「admin」或「team_admin」

🖱️ 進入步驟

  • 在主選單找到「管理員專區」按鈕
  • 按鈕為紫色，位於主選單下方
  • 點擊後進入管理員專區主頁
  • 進入後會看到各項管理功能選項

2️⃣ 團隊資訊管理
首頁 → 管理員專區 → 團隊資訊

🏢 團隊基本設定

  • 點擊管理員專區中的「團隊資訊」按鈕
  • 進入團隊資訊管理頁面
  • 可以設定團隊名稱和球場資訊
  • 上傳球場照片增加視覺效果

📝 填寫團隊資料

  • 場地名稱 - 輸入球場或團隊名稱
  • 場地照片 - 上傳代表性照片
  • 球隊介紹 - 詳細描述團隊特色、比賽週期、開放時間、停車方式等
  • 收費政策 - 設定收費方式，例如每季收費或每年收費金額
  • 聯絡方式 - 填寫聯絡人和聯繫方式

3️⃣ 會員管理系統
首頁 → 管理員專區 → 會員管理

👥 進入會員管理

  • 在管理員專區點擊「會員管理」按鈕
  • 系統會顯示當前團隊的所有會員
  • 您可以看到會員編號、姓名、聯絡方式等資訊
  • 右側有操作按鈕可以編輯或刪除會員

🔧 主要功能

  • 會員總數統計 - 顯示當前團隊的所有會員數量
  • 查看會員編號、姓名、聯絡方式等資訊
  • 右側有操作按鈕 - 可以編輯或刪除會員
  • 重複名字檢查 - 系統會提醒重複的會員姓名

⚠️ 重複名字檢查功能

  • 當新增會員時，系統會自動檢查同團隊中是否有相同姓名
  • 如果發現重複姓名，系統會顯示警告訊息
  • 建議在重複的姓名後加上識別資訊，如「張三(北)」、「張三(南)」
  • 或使用全名避免重複，如「張三豐」vs「張三峰」
  • 確保每位會員都有唯一的識別名稱，避免混淆
4️⃣ 單一會員建立
首頁 → 管理員專區 → 會員管理 → 新增會員

➕ 新增單一會員

  • 在會員管理頁面點擊「新增會員」按鈕
  • 系統會彈出新增會員的表單
  • 會員編號會自動產生（如：S0002）
  • 填寫完成後點擊「新增」按鈕

📝 必填欄位說明

  • 會員編號 - 系統自動生成，格式：團隊代碼+數字
  • 姓名 - 必填，會員的真實姓名
  • 電話 - 選填，聯絡電話
  • 加入日期 - 預設為當天，可調整
  • 級數 - 選填，可輸入數字或留空
  • 備註 - 選填，其他相關資訊

5️⃣ 批次會員建立
首頁 → 管理員專區 → 會員管理 → 批量新增會員

📋 批量新增功能

  • 在姓名欄位可以貼上多個姓名（每行一個）
  • 系統會自動為每個姓名建立一列
  • 會員編號會自動依序生成
  • 加入日期會統一設定為當天
  • 一次可以新增多位會員，大幅提高效率

🔧 操作步驟

  • 準備會員姓名清單（每行一個姓名）
  • 在新增會員表單的姓名欄位貼上清單
  • 系統自動展開為多列資料
  • 檢查各欄位資訊是否正確
  • 點擊「批量新增」或逐一確認

批量新增會員實際介面

此介面顯示批量新增會員的完整操作畫面，可以清楚看到姓名列表的自動展開功能

💡 批次建立優勢

  • ✅ 效率提升 - 一次建立多位會員
  • ✅ 自動編號 - 系統自動分配會員編號
  • ✅ 統一設定 - 加入日期等資訊統一處理
  • ✅ 錯誤檢查 - 自動檢查重複和格式
⚠️ 重要提醒
  • 管理員權限：只有admin或team_admin角色才能進行會員管理
  • 會員編號唯一性：每個會員編號在系統中必須是唯一的
  • 資料備份：建議定期備份會員資料，避免意外遺失
  • 團隊隔離：每個團隊只能看到和管理自己團隊的會員
  • 即時生效：新增的會員可以立即登入使用系統
